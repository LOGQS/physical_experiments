## Purpose

Implementations and experiments for robotics and physical systems, mainly for learning.

## What's here

- **Reimplementations** of known methods to learn
- **Novel ideas**  that I'm working on, though some half-baked, some more serious, hard to determine what's actually novel most of the time
- **Experiments** with notes on what worked, what didn't, and why I think that happened, generally small scale

## Structure

Will probably change based on what makes sense for a given context, but roughly: each directory is a category or a single architecture/concept. Inside you'll find the implementation/s, a notes file with my observations and experiment logs when I ran actual tests. Not everything has been validated, some ideas are just written down before I forget them.

## Status

Active. I add things when I have time between uni and other projects. This is a new line of work that I will start.
